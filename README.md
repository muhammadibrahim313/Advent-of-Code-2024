## 🚀 Quick Setup

### Clone & Explore

```bash
# Clone the repository
git clone https://github.com/muhammadibrahim313/Advent-of-Code-2024.git

# Navigate to a day's challenge
cd Day1

# Open the Jupyter Notebook
jupyter notebook Day1.ipynb
```
